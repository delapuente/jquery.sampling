jQuery.sampling plugin¶

Setup¶

You can configure the jQuery.sampling() plugin by calling jQuery.sampling() on the jQuery object (on $ for newbies).

jQuery.sampling([options])¶

Establishes initial options such as the sample class or new flags. See jQuery.sampling() options for further information.

Arguments:
  • options (object) –

    a configuring object which defaults to:

    {
    sampleClass: '_s',
    defaultFlags: {
        ensureText:false,
        clearSamples:true
    }
}

jQuery.sampling() options¶

sampleClass (defaults to _s)
class to mark HTML elements as samples. These elements will be gathered by using gather().
defaultFlags
see Instantiation flags for more information about these options.

Instantiation flags¶

ensureText (defaults to false)
during instantiation, content is set to each element by using jQuery.text() instead of jQuery.html() as it’s by default.
clearSamples (defaults to true)
this option removes all nested samples inside the sample being instantiated.

Plugin API¶

Following jQuery Namespacing convenions, all these methods must be called by using:

jQuery.sampling(<method_name>, ...);

I.e, to call option() to set the value of parameter sampleClass to _sample you should do:

jQuery.sampling('option', 'sampleClass', '_sample');
gather()¶

Applies on jQuery object. Detaches all sample elements marked by the sample class (by default _s) and returns a collection of them.

Returns:sample collection with all collected samples.
select(selector[, noCache=false])¶

Applies on sample collection returned by gather(). Selects the proper sample inside the sample collection. The method is cached by the selector so succesive calls to this method have improved performance.

Arguments:
  • selector – a valid jQuery selector to look for it inside the sample collection.
  • noCache (boolean) – if true, avoid caching by the selector and the query is re-runned. Defaults to false.
Returns:

a jQuery sample object with the sample. Avoid to modify this object because it’s the sample itself. Use jQuery.clone() or use new() to build a new copy.
option(optName)¶

Get value of the optName option.

Arguments:
  • optName (string) – name of the option to get.
Returns:

value of the option specified by optName parameter.
option(optName, value)

Set the optName option to value.

Arguments:
  • optName (string) – name of the option to set.
  • value – value to set.

Instantiating samples¶

Sample instantiation is the most interesting part of the plugin. Instantiate refers to clone the sample and customize it.

The task can be performed by calling to new() in several ways. The most important concept here is customizing item.

Customizing items¶

When a sample is instantiated it is customized by a customizing item. This is an ordinary object where keys are valid jQuery selector strings and values are mixed contents.

During customization process, each key of the customizing item is searched inside the sample and its content is set with the value.

In instance, look at this customizing item:

{
    '.id':1,
    '.name':'Tie Fighter',
    '.type':'Starfighter',
    '.affiliation':'Galactic Empire'
}

Now look at this HTML snippet:

<tr class="_s">
    <td class="id">sample_id1</td>
    <td class="name">sample_name1</td>
    <td class="type">sample_type1</td>
    <td class="affiliation">sample_affiliation1</td>
</tr>

Previous tr sample, after instantiating with the previous item should look like:

<tr>
    <td class="id">1</td>
    <td class="name">Tie Fighter</td>
    <td class="type">Starfighter</td>
    <td class="affiliation">Galactic Empire</td>
</tr>

Following, there is the documentation about the different ways of calling new():

new()¶

Applies on a jQuery sample object returned by select().

Returns:a clone of the current sample without the sample class.
new(map_generator[, flags={}[, ...]])

Applies on a jQuery sample object returned by select(). Successively calls map_generator using the returned value as the customizing item (see Customizing items) to instantiate the sample. Instantiation process ends when null is returned.

Arguments:
  • map_generator (callback) –

    the callback signature should be as follows:

    map_generator(index[, ...])

    It receives the number of call (first time, 0; second time, 1; third time, 2...) and all other parameters passed to new() method.

    It should return a customizing item to instantiate the sample or null to end the instantiation process.

    Inside the callback, this refers to the sample’s copy being instantiated.

    Note

    Remember passing flags parameter (you can pass an empty object {} if you cannot change any flag) if you want to add parameters to the map_generator.

  • flags (object) – flags to customize the sample’s instantiaion. Default flags can be set by using jQuery.sampling() or option(). Flags are explained in Instantiation flags.
  • ... (mixed) – parameters for the map_generator.
Returns:

a jQuery object, clone of the current sample, without the sample class.
new(customizing_array[, flags={}[, ...]])

Applies on a jQuery sample object returned by select(). Generates as much instantiations of the sample object as elements in the array.

Arguments:
  • customizing_array (array) –

    The array containing the customizing items to instantiate the sample. Items inside the array can be of one of these types:

    • object: the object must be a customizing item (see Customizing items)
    • function: the function is call under same conditions of map_generator in the previous new() method but it must return a customizing item.
    • rest of the cases: the item will be the content of the instance.

    Note

    Remember passing flags parameter (you can pass an empty object {} if you cannot change any flag) if you want to add parameters to the functions inside the array.

  • flags (object) – flags to customize the sample’s instantiaion. Default flags can be set by using jQuery.sampling() or option(). Flags are explained in Instantiation flags.
  • ... (mixed) – parameters for the map_generator.
Returns:

a jQuery object, clone of the current sample, without the sample class.

Tip

If you want to pass an array of just one item, and this item is not a function, then ignores the array and pass only the item.
